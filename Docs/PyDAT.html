Syntax^

Every decompiled DAT file uses the same syntax, just with different keywords. Each entry starts with an ID header, then is followed by a list of all the settings for that id:
Code 
TypeID ID:
    Data	Value
    Data	Value
    ...
TypeID and the Data/Value pairs depend on which type of DAT file it is. The different types are:
  • UnitID
  • WeaponID
  • FlingyID
  • SpriteID
  • ImageID
  • UpgradeID
  • TechID
  • SoundID
  • PortraitID
  • MapID
  • OrderID
ID is the ID of the entry in the DAT file (for example UnitID 0 is Terran Marine). There are only three types of values in a DAT file:
  • Quantity (like MineralCost or BuildTime)
  • Flags (like SpecialAbilityFlags)
  • Data entry (either from another DAT file, TBL file, or one of a list of values, given in the Reference)
When decompiling entries from a DAT file, data entry values will always have a comment at the end, telling which type of data entry it is and giving additional information on the value. The only values that are flag sets are the values used by data keywords ending in Flags (like SpecialAbilityFlags). Everything else is a quantity.

The only entries that take flags are SpecialAbilityFlags, StarEditGroupFlags, and StarEditAvailabilityFlags in units.dat, and TargetFlags in weapons.dat. Restrictions^

Some data entries are restricted to certain ID's in the DAT files, they are:
  • units.dat entries 106-201 are the only unit entries that have the data entries: Infestation, AddonHorizontal, AddonVertical
  • units.dat entries 0-105 are the only unit entries that have the data entries: ReadySound, PissSoundStart, PissSoundEnd, YesSoundStart, YesSoundEnd
  • sprites.dat entries 130-516 re the only sprite entries that have the data entries: HealthBar, SelectionCircleImage, SelectionCircleOffset


Example Usage^

Decompiling unit id 99 (Samir Duran), then recompiling back into the same dat file:
Command Line 
python PyDAT.pyw -n 99 units.dat
  ..edit dat..
python PyDAT.pyw -b units.dat -c units.txt
Decmpiling every entry in mapdata.dat, then recompiling:
Command Line 
python PyDAT.pyw -m MyMod\mapdata.dat
  ..edit dat..
python PyDAT.pyw -m -c MyMod\mapdata.dat
